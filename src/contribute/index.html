Follow these steps to upload photos and contribute to the repository.

Step 1: Fork the Repository

Click the Fork button on the GitHub repository to create your own copy.

Step 2: Clone the Repository

git clone https://github.com/vishnu1100/ForzaLab.git

Navigate to the project directory:

cd ForzaLab

Step 3: Upload Your Images

Place your images inside the src/uploads/images/ folder.

Ensure your images are in JPG, PNG, or GIF format.

Step 4: Commit and Push Changes

git add .
git commit -m "Added new images"
git push origin main

Step 5: Create a Pull Request

Go to the GitHub repository, click Pull Request, and submit your changes.

Step 6: Get Merged!
